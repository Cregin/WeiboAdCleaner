# SinaWeiboAdCleaner
1. 去除微博顶部广告
2. 去除微博列表中广告

# Requirements
1. iOS Toolchain
2. Theos
3. iOS Version >= 8.0

# Usage
1. 修改 Makefile `THEOS_DEVICE_IP = 你的设备IP`
2.  `make package install`
