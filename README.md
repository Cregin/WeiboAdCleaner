# SinaWeiboAdCleaner
去除微博顶部广告、
去除微博列表中广告

# Requirements
iOS Toolchain
Theos

# Usage
1. 修改 Makefile `THEOS_DEVICE_IP = 你的设备IP`
2.  `make package install`
